# Safari Booking Platform - Frontend Application

This directory contains the client-side user interface for the Safari Booking Platform. It is a Single Page Application (SPA) developed using React.js and Bootstrap.

## System Requirements

* Node.js (LTS version recommended)
* npm (Node Package Manager)

## Installation

1.  Open a terminal and navigate to the `frontend` directory.
2.  Install the required dependencies by running:
    ```bash
    npm install
    ```

## Execution

To start the development server:

```bash
npm start
